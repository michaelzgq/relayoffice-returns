## Acceptance

Acceptance means:

- homepage loads
- login page loads
- owner account signs in successfully
- owner can access the workspace
- `Workspace Access` page is accessible

## Support Boundary

This quote includes Dossentry package guidance and the agreed support window only.

It does not include:

- ongoing customer infrastructure administration
- emergency on-call support
- custom feature work
- unlimited user training

## Deployment Responsibility Split

- internal credential custody

- Dossentry package guidance
- initial deployment workflow
- first-time acceptance validation
