# Network Promotion 30-Day Plan

最后更新时间：2026-04-08 America/Los_Angeles

## Goal

未来 `30` 天内，不靠付费广告，用网络推广拿到：

- `5` 个有效 discovery calls
- `1` 个付费 review
- `1` 个真实 pilot 候选

这份计划默认目标不是“做品牌曝光”，而是：

**让对的人看到一个足够具体的问题，并愿意和你对话。**

## Starting Point

### Current Advantages

- 产品已上线，可直接演示
- 有一个可直接演示的 live demo URL
- 已有 `Brand Review Link`
- 已有相对清晰的定位：`warehouse-side evidence + decision-control layer`

### Current Constraints

- 还没有真实客户案例
- 还没有明确验证 `refund authority` 在谁手里
- 市场教育不能靠广告烧出来
- 当前阶段最大瓶颈不是流量，而是高质量对话数量

## Core Principle

现在不要追求：

- 大量访客
- 大量注册
- SEO 大词排名
- 品牌“看起来很大”

现在只追求：

**让被品牌 challenge 过 returns handling 的人看到你。**

## Offer Stack

网络推广不能直接卖“软件订阅”，要卖阶梯式动作。

### Offer 1: Low-Friction CTA

`Book a 20-minute workflow review`

用途：

- 让第一次接触的人愿意对话
- 降低“我现在就要买软件吗”的心理压力

### Offer 2: Paid Entry Offer

`Returns Decision Support Review`

建议价：

- `Pilot price: $300`

用途：

- 验证痛点和付费意愿
- 获取真实 workflow 和真实 objection

### Offer 3: Pilot

`Brand Review / decision-control pilot`

用途：

- 从“看 demo”切到“跑真实 case”

## Recommended Channel Mix

未来 `30` 天只做这 `5` 类，不扩散。

### 1. Website

目的：

- 不是 SEO
- 是接住所有 DM、邮件、评论流量

网站要只完成 `3` 件事：

1. 让人知道你不是 shopper portal
2. 让人知道你解决的是 warehouse-side disputes
3. 让人知道下一步是约 review 或看 demo

### 2. Founder-Led LinkedIn

目的：

- 最容易直接触达 owner / ops manager
- 既能冷启动触达，也能积累认知

打法：

- `5` 个精准连接请求 / 天
- `5` 个 DM / 天
- `3` 条内容帖 / 周
- `10` 条高质量评论 / 周

### 3. Cold Email

目的：

- 对 LinkedIn 不回的人做第二触点
- 跟网站表单/名单形成闭环

打法：

- `5-10` 封 / 天
- 每封都带一句个性化 opening

### 4. Fulfill / Directory Presence

目的：

- 占住“3PL 正在找工具”这类高意图入口
- 不靠大流量，靠高 intent

阶段建议：

- 先上免费或最低成本档
- `14` 天内观察是否带来高质量线索

### 5. Reddit / Community Research Posts

目的：

- 不是卖产品
- 是验证痛点语言、收集原始表达

规则：

- 发问题帖，不发广告帖
- 帖子只讨论 workflow，不贴产品链接

## What Not To Do

未来 `30` 天先不要做：

- Google Ads
- Meta Ads
- Product Hunt
- 大规模 SEO 外包
- 同时面向所有物流、仓储、电商人群
- 首页就强调 `AI / self-hosted / lifetime deal`

原因：

- 这些都比“找对人”更耗钱、更慢、更容易失焦

## ICP For Promotion

### Primary ICP

- 多品牌 3PL
- 服务 `5+` 个 DTC 品牌
- 位于 California / Texas / New Jersey 优先
- 每月 returns 有一定量
- 过去 `6` 个月被品牌 challenge 过 returns handling

### Secondary ICP

- 多品牌 operator / brand aggregator
- 自有仓库
- buyer 和 user 接近

### Channel Strategy By ICP

- `3PL`：LinkedIn + cold email + Fulfill
- `multi-brand operator`：LinkedIn + cold email

## Message Hierarchy

不要先讲：

- “我们功能很多”
- “我们也支持 AI”
- “本地部署”
- “终身买断”

先讲：

### Primary Message

`When a brand challenges how your warehouse handled a return, your team should be able to send one link with the evidence, timeline, and recommendation.`

### Secondary Message

`We standardize brand-specific warehouse-side return rules without asking your team to run the workflow from SOP docs, spreadsheets, and chat threads.`

### Tertiary Message

`No hardware. No heavy rollout. Start from a browser.`

## 30-Day Execution Plan

| Week | Objective | Tasks | Output | Exit Criteria |
|---|---|---|---|---|
| Week 1 | 把对外资产补齐 | 完成官网单页、Loom、sample case、目标名单 `50` 家 | 可被分享的最小销售资产 | 有人看完 `60` 秒内能懂你卖什么 |
| Week 2 | 开始精准触达 | 每天 `5` 封邮件、`5` 条 LinkedIn DM、`2` 次 follow-up | 第一轮真实回复和 objection | 至少 `2` 个正向回复 |
| Week 3 | 调整话术并继续打 | 根据回复优化文案；发 `3` 条 LinkedIn 内容；上 Fulfill listing | 更准的 messaging | 拿到 `3` 个有效 calls |
| Week 4 | 收口成 review / pilot | 跟进、demo、卖 paid review、整理 objections | 第一个付费验证 | 至少 `1` 个 paid review 或强 pilot 信号 |

## Weekly Cadence

### Monday

- 更新目标名单
- 发第一轮触达
- 发一条 problem-first 帖子

### Tuesday

- 做 DM follow-up
- 评论行业帖子
- 复盘上周回复语言

### Wednesday

- 发第二条内容帖
- 跑 discovery calls
- 更新 tracker

### Thursday

- 再打一轮冷邮件
- 给有互动的人发 Loom

### Friday

- 发第三条内容帖
- 总结 objections
- 决定下周主话术

## KPI

只看这 `8` 个数：

1. 精准触达总数
2. 回复率
3. discovery calls 数
4. 有效 calls 数
5. 付费 review 数
6. 有人明确承认被品牌 challenge 的次数
7. 有人明确说 `Brand Review Link` 有价值的次数
8. authority answer
   - `3PL controls`
   - `brand controls`

## Decision Rules

### Keep Going

- `50-75` 次精准触达内拿到 `5` 个有效 calls
- 至少 `1` 个人愿意付 review fee
- 至少 `2` 个人明确承认存在 returns disputes / proof challenge

### Pivot

如果有效对话都在说：

- 退款决定不在 3PL 手里
- 但品牌争议和证据需求很强

则主叙事转成：

- `evidence readiness`
- `brand review`
- `shareable case record`

### Kill

如果打完：

- `75` 次精准触达
- `5` 个以上有效对话

仍然没有人承认：

- 被品牌 challenge
- 证据难找
- rule drift
- 处理高风险 case 很乱

那不是推广问题，是定位问题。
