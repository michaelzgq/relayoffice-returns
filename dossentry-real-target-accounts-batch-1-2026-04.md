# Dossentry Real Target Accounts - Batch 1

Last updated: 2026-04-13 America/Los_Angeles

## Goal

This is the first real outbound batch for Dossentry.

It is not a theoretical ICP list.
It is a ranked list of real companies with public evidence that they already do returns, reverse logistics, return processing, or warehouse-side exception handling.

Important boundary:

- I only included companies I could verify from public sources
- when I could not verify a named decision-maker from a public source, I listed the target role instead of inventing a person
- this list is optimized for fast founder-led outreach, not enterprise logo collecting

## How To Use This File

For each account:

1. find the role listed here
2. send the `compare` link first
3. use the `utm_term` assigned here
4. only send the PDF after a reply, a click, or a real follow-up

Default first-touch link pattern:

```text
https://dossentry.com/compare/generic-inspection-apps?utm_source={{source}}&utm_medium={{medium}}&utm_campaign=2026q2_wk1_message_validation&utm_content=compare_batch1&utm_term={{term}}
```
